Welcome developer

This is the unofficial API for the SNCF/NMBS in Belgium. If you want to implement train schedules in your application you're at the right place.

Usage

1. List all stations

Request: irail.be/api/stations.php

Response: 

        <stations>
            <station>
                GENT SINT PIETERS
            </station>
            ...
        </stations>

2. Trainschedule

Request: http://www.irail.be/api/trains.php?to=STATION1&from=STATION2 |||OPTIONAL: &date=01-01-10&time=15:40&results=4&lang=NL

STATION1 and 2 should be all capitals, like in the stations.php file

Response: 

        <connections>
            <connection>
                <departure>
                    <station>
                        HARELBEKE
                    </station>
                    <time>
                        17:24
                    </time>
                    <date>
                        03/08/10
                    </date>
                </departure>
                <arrival>
                    <station>
                        GENT SINT PIETERS
                    </station>
                    <time>
                        17:51
                    </time>
                    <date>
                        03/08/10
                    </date>
                </arrival>
                <duration>
                    0:27
                </duration>
                <delay>
                    1 or 0
                </delay>
            </connection>
            ...
        </connections>
